This is a simple demonstration of using jsPlumb to display hierarchical information

On this page we set our connectors to gray and specify an endpoint of radius 9. Each connection has an Arrow overlay indicating direction, and paints itself orange on mouse hover

The Bezier curve used on this page has a 'curviness' of 50; See the documentation for examples of how to set the Bezier curviness.

This demonstration uses jsPlumb 1.3.10, jQuery 1.7.1 and jQuery UI 1.8.13.
